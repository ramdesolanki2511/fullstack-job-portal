# Job Portal Management System

## Features

- JWT Authentication
- RBAC
- Candidate Dashboard
- Recruiter Dashboard
- Admin Dashboard
- Resume Upload
- Cloudinary Integration
- Email Notifications
- Search & Pagination
- MongoDB Atlas

## Tech Stack

- Next.js
- Node.js
- Express.js
- MongoDB
- JWT
- Cloudinary
- Nodemailer
